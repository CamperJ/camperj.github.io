Basics

Required materials:

  • Resume or curriculum vitae (CV)
  • Cover letter (sometimes optional, but generally good to have)
  • List of References
Strategy:
Job offers = Applications * (Automated screens pass rate) * (HR screen pass rate) * (Interviews pass rate) ^ (n interview rounds) * (references pass rate) * (background check pass rate)
As shown in this equation, unfortunately any principal factor can wipe out the others if its value is 0, for example if a candidate gets through all of the interviews but then doesn't pass the background check.
The goal then is to work on each successive factor until it is passed, and then work on the next one, until all factors are >0.
